# Python CLI template

Python CLI application template.

## Requirements

* Python 3.6 or later

## Minimal

Minimal CLI template.

```sh
python minimal
```

## Simple

Simple CLI template.

```sh
python simple hello
```

## Simple project

Simple CLI project template.

```sh
python simple-project hello
```

## Project

CLI project template.

```powershell
# PowerShell
$Env:PYTHONPATH = './project/src'
python project/src/proj/command hello
```

```bash
# bash
export PYTHONPATH='./project/src'
python project/src/proj/command hello
```
